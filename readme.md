# Carpool Manager

### Features to be implemented

- each person can get a list of when they are driving and who they are driving
- each person can get a list of which carpools are driving them and who will be driving them
- if a driver already has a carpool scheduled, when choosing driving availability for a new group, remind them of their other carpools
- can send users messages via phone or email
- can send reminders for rides
- any user can make a carpool group (they will be the admin)
- admin sets the trip times at the time of the carpool group creation
- when creating an account (family) all people on the family account are created together
- people who join the carpool will select which times they can be drivers and which times they require pickups
- IMPORTANT: need to update schema to include the extra address for each carpool (original pickup location or final drop-off location)
-
